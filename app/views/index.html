Design pattern
Task lists

Help users understand and complete all the tasks involved to achieve their goal; become a childminder, learn to drive, start a business.

As the pattern is bigger than a web-page, we've used an example service – become a childminder – to show it in use.

Starting points for users:

  • Single task to list of tasks, non-linear
  • List of tasks to single tasks, linear
